PulsePoint is a full-stack web application designed for MBA students and business professionals. It aggregates news from 20+ top-tier sources including CNBC, Bloomberg, BBC, and Indian financial dailies.

## Features
- **Curated Feeds**: Finance, Macro Policy, Strategy, Startups.
- **India Focus**: Dedicated section for Indian markets and economy.
- **Smart De-duplication**: Removes duplicate stories across feeds.
- **Market Snapshot**: Live data for Sensex, Nifty, S&P 500, Gold, etc.
- **Premium Design**: Editorial aesthetic with dark mode elements.

## Local Setup
1. Install Node.js.
2. Run `npm install` to install dependencies.
3. Run `node server.js` to start the server.
4. Open `http://localhost:3000`.
